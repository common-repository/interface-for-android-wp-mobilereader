== Description ==
Interface for Android WP MobileReader.
app languages actually: EN, DE
I not assume liability or warranty for possible damage of you webpage.

You website content on Android smartphone. Ready-to-use WP-Plugin and Android app.
Do not lose your user on smartphones!

Install plugin on your WordPress.
Your user get the app from Play Store, add URL your page to app`s sitelist and can instantly read your content.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/interface-for-android-wp-mobilereader` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Activate editing on options page if needed
4. INSTALL APP \'WP MobileReader\' FROM GOOGLE PLAY STORE ON YOUR ANDROID SMARTPHONE: 
http://play.google.com/store/apps/details?id=de.trames.wwclient
5. Add you URL to sitelist in app.
